# 51-MCU-Music-Tool
帮助大家在学习51单片机蜂鸣器的时候，方便播放出音乐。
51单片机蜂鸣器音乐转码工具 (MCU Music Tool)
这是一个专为 51单片机 开发设计的上位机辅助工具。它可以将标准的“简谱文本”快速转换为单片机定时器可执行的 C 语言数组，配合无源蜂鸣器实现高质量的音乐播放。

🌟 功能亮点
可视化 GUI 界面：基于 Python Tkinter 开发，操作直观。

动态 BPM 设置：支持 60~200 BPM 速度调节，适配不同曲风（如《千本樱》等快歌）。

智能数据转换：自动计算 16 位定时器初值（TH0/TL0），无需查表。

一键精简复制：支持只复制数组内部的十六进制数据，方便直接粘贴到 Keil。

溢出保护：自动处理长音符导致的毫秒数溢出问题（使用 unsigned int 存储）。

🚀 程序使用步骤
1. 上位机转换
运行 dist/jianpu_to_c51.exe（或运行源代码 python jianpu_to_c51.py）。

在输入框中粘贴简谱字符串。

根据曲目需求设置 BPM（歌曲速度）。

点击 “生成 C 代码”，并在下方查看结果。

点击 “一键复制数据” 提取数组核心内容。

2. 下位机集成
在 Keil 中定义数组：unsigned int code Music_Data[] = { /* 粘贴处 */ };。

确保硬件连接：蜂鸣器驱动电路连接至单片机 P2.5 引脚（或根据代码修改 sbit）。

编译并烧录至 51 单片机。

🎵 旋律输入格式
程序采用 “音符:时值” 的简谱录入方式，多个音符间以空格分隔。

音符前缀：L (低音)、M (中音)、H (高音)、0 (休止符)。

音符数字：1 至 7 代表 Do 至 Si。

时值数字：4 代表四分音符，8 代表八分音符，16 代表十六分音符。

示例（《春日影》片段）：
M3:8 M4:8 M5:4 M5:4 M3:8 M5:8 H1:4 M7:4

⚠️ 重要提醒：数组容量与内存限制
51 单片机（如传统的 AT89C51/52）的资源非常有限，在使用本工具时请务必注意：

Flash (Code) 空间限制：标准的 51 单片机通常只有 8KB 的 Flash 空间。本工具生成的每个音符占用约 6 字节。

建议单首曲目不要超过 100 个音符，否则可能导致 Keil 报错 code space overflow（代码空间溢出）。

Delay 阻塞：当前的播放逻辑使用 Delay 函数，这会阻塞主循环。在播放期间，按键扫描可能会变得不灵敏。

硬件保护：请确保使用无源蜂鸣器。建议在 P2.5 引脚输出时配合三极管驱动电路，避免直接驱动导致引脚电流过载。

🛠️ 开发环境
PC端：Python 3.14+, PyInstaller

单片机端：Keil uVision5, C51 编译器
